GitHub commands




1) To insert a new text file in the repository through local


insert -> add -> commit -> push 

git add filename.txt

git commit -m "filename.txt added"

git push origin main





2) To modify the file follow the same process


modify -> add -> commit -> push 





3) To delete the file follow the same process


delete -> add -> commit -> push
